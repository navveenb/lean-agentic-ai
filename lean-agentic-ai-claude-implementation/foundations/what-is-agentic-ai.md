# What Is Agentic AI?
> From reactive to goal-oriented systems.

Agentic AI systems don't just respond to prompts — they reason, plan, use tools, maintain state, and take actions to achieve goals. Unlike a traditional LLM call (input → output), they operate in loops: perceive → reason → act → observe → repeat.

Each capability has a cost: reasoning = extra tokens, planning = additional LLM calls, tool use = execution time, memory = storage costs, autonomy = risk of unchecked resource consumption. Lean design makes them intentional, bounded, and efficient.
