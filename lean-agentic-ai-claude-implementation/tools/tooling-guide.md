# Tooling Guide
**Cost**: LiteLLM (unified API + cost tracking), Helicone (observability), LangSmith (tracing)
**Caching**: GPTCache (semantic), Redis (exact-match), provider prompt caching
**Carbon**: Electricity Maps, WattTime, CodeCarbon, Cloud Carbon Footprint
**Observability**: LangFuse, Arize Phoenix, OpenLIT
**Providers with lean features**: Anthropic (prompt caching, Haiku/Sonnet/Opus tiers, batch API), OpenAI (prompt caching, Mini/4o/o1 tiers), Google (context caching, Flash/Pro tiers)
